Le dossier source accueille les documents utiles à l'intégration. Il peut s'agir du code à reprendre, des maquettes à intégrer ou encore des textes à publier, suivant le nombre d'éléments et surtout de maquettes. Il peut être utile d'avoir recours à des sous-dossier.

Css: Contient le style, police, les images thématique du site (ex: Noel, solde etc..)
Un sous dossier "Theme" contiendra les images thématiques du site

Image Contient les images de contenu (picto, menu etc...)

script: contient les différents fichier Javascript

règles de nommage des fichiers:
employez des noms anglais pour tous les éléments, car la V 2 sera en multilingue.
Indiquez tous les noms au singulier. 
Utilisez uniquement les traits d'union au lieu des underscore.
Donner des noms pertinents aux éléments.
Utilisez les classes et identifiants comme nom d'image.
Privilégiez les intitulés cours.
Différencier les premiers éléments d'une liste plutôt que le dernier
